DataBinding

Databinding is how the typescript/javascript code communicates with the HTML template.
There are 4 forms of data-binding:

1. String Interpolation

Allows strings, or objects that can be implicitly converted to Strings outputted to the view.
usage: output the value within double curly braces in the HTML template i.e. {{ '{' }}{{ '{' }} myVariable {{ '}' }}{{ '}' }}
MyVariable can be a variable, a hardcoded string, a method that returns a string, a value that can be converted to a String (implicilty done by Angular).
For Example, a number will work because Angular will know to implicitly convert this into a String.
Within the double curly braces, you cannot put conditional statements.

> Usage:
Server with {{ '{' }}{{ '{' }}serverID {{ '}' }}{{ '}' }} is {{ '{' }}{{ '{' }}getServerStatus(){{ '}' }}{{ '}' }}
> Example: 

    // variables declared in the ts file
    myImplicitString = 'implicit string'; 
    myExplicitString: string = 'explicit string'; 
    myNumber: number = 10;

    getStringFunction() {{ '{' }}
        return "String returned from String Function";
    {{ '}' }} 

    // using the variables in the html template
    Implicit String : {{ '{' }}{{ '{' }}myImplicitString{{ '}' }}{{ '}' }} 
    Explicit String : {{ '{' }}{{ '{' }}myExplicitString{{ '}' }}{{ '}' }} 
    Explicit number : {{ '{' }}{{ '{' }}myNumber{{ '}' }}{{ '}' }} 
    Function returning a String : {{ '{' }}{{ '{' }}getStringFunction(){{ '}' }}{{ '}' }}
> Output: 

    Implicit String : {{myImplicitString}} 
    Explicit String : {{myExplicitString}} 
    Explicit number : {{myNumber}}
    Function returning a String : {{getStringFunction()}}

2. Property Binding

Allows Angular to set the value of a property on an HTML element dynamically.

> Usage
<blah [property]="value">
> Example: 

    //variable declared in ts file
    myPropertyBinding: boolean = true;

    // buttons used in html template with/without property binding
    <button class="btn btn-primary" disabled="false">WithOut Property Binding</button> 
    <button class="btn btn-primary" [disabled]="myPropertyBinding">With Property Binding</button>
> Output:

3. Event Binding

Allows Angular to set the value of an event on an HTML element dynamically.
To bind to HTML element events in Angular, put the event in circle brackets and remove the 'on' part of the event name i.e. (click) instead of onclick

> Usage 
<blah (input)="expression">
// Use $event to retrieve the properties of the event
<blah (input)= "onUpdateServerName($event)">
> Example: 

    //variable declared in ts file
    <button class="btn btn-primary" (click)="onEventBindingIncreaseNumber()">Increase Number</button> {{ '{' }}{{ '{' }}eventBindingNumber{{ '}' }}{{ '}' }}
> Output:

{{eventBindingNumber}}

This example shows how to use the $event to retrieve the value from the event and bind it back to a variable {{eventBindingString}}
